# FAST-Dynamic-Vision

**FAST-Dynamic-Vision** is a system for detection and tracking dynamic objects with event and depth sensing.

- [ ] This paper has been submitted to [IROS 2021](https://www.iros2021.org/) and will be open-source soon.
- [ ] Related paper is avaliable at: [arXiv](https://arxiv.org/abs/2103.05903).
